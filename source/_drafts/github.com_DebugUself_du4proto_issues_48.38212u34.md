## 方法
Google 大法好
- 尽可能多的英文关键词搜索(专业词汇)
- +/OR/-, 分别对应逻辑与/或/非，其中逻辑与+，可用空格代替
- 尽量多个keywords 一起搜，除非想搜的是什么特定专有名词
- 除了error msg，不要一整句话一起搜，只提取名词和重要动词
- filetype:xxx, 搜索特定格式的文件。
- site:xxx,搜索指定网站的资源
- 专业的词
- 满屏都是, 说明有影响力, 不要视而不见.

		- crawler vs. spider

	github crawl -\> scrapy(most stars)
