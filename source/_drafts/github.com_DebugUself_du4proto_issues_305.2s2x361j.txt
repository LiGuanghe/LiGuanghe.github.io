## 大妈反馈
- JS 形式不习惯
- python 带来的正确思路, 无论程序用什么语言写的, 最终是把人的思想表述给计算机. 其他语言对形式比较模糊. python 根据缩进符把形式固定下来. 语法不习惯的时候就是难点和技术点. 猜测所有形式之前, 先把程序跑通, 有实时调试的空间, 先根据你的想法, 改代码, 跑一下.
- JS 的设计就是立刻执行, 只能用回调函数还限制请求/等待和处理. 强行把立刻执行变成不立刻执行. 
- 微信小程序是 JS 的子语言, 必须这样写. 有其他资源从形式化改进 JS. 所有变量都是全局变量, 代码都是全局代码.
- 尽量调试, 这样可以快速实现你对语言的理解. 
- 搞清楚, 用微信小程序做什么, 先做一遍, 找到用什么形式能 ok, 不知道再查. 
- 如何学一个新语言

- 为什么用 JS , 如何和 python 对接?
    - 因为 JS 是所有浏览器内置的, 不用另外安装其它运行时环境,即所有手机上唯一明确相同的,只有 JS 的运行时
微信小程序是种定制的 JS 环境本身是封闭的,
但是, 只要你有后台服务, 那么总是有标准的http 协议接口的
服务端就可以用 python 了
即通过 RESTful 接口来融合所有异语言系统
