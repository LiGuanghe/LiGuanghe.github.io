##  大妈反馈
  + 知道给输出分类,应将时间账单与输出结合汇报.
  + 探索姿势, pyenv小鹤用330min, 大妈30min, 
    + 但大妈形成这个概念已经是十几年的了. 从 windows开始, 当时已经感觉到多环境, 但当时还没有 python3, 是几个 python2的不同版本, 后来遇到 pyenv, 才顺理成章的使用了.
    + 高手用新工具快是因为这个问题之前已经想过很久, 只是自己没遇到好的解决办法, 所以在遇到好的工具时立刻知道解决了什么痛点,可以理解基于什么原理, 也可以根据实际情况, 使用适合痛点的工具. 需求才能产生工具和概念. 
    + 要回答: 现在的问题是什么? 解决问题的最好方式是什么? 
    + 随着自己的经历积累后, 问题困难点升级, 解决不同复杂度的问题也要随之升级.这是反常识的点, 与日常教育不同, 日常教育希望告诉你先解决核心问题, 之后其他的问题都不是问题. 但这是不可证伪的. 
    + 这是知道自己不知道的案例. 
  + 德语/英语
      + 用的多了, 自然知道. 
      + 1checker 是语法工具, 优点是有实时的反馈, 惯用法上及时反馈. 当你先说了自己的中式英语后,软件不仅仅帮你改语法错误, 而是提供错误报告, 比较好的方法是, 每次写之前, 看看上一次写作报告, 先独立解决掉看看, 然后再用这个软件检查, 这是正向反馈.而不是知道正确答案就可以了.
      + 英文 blog, 同样的内容其实要写五遍:
        + 第一遍即时录音. 
        + 第二遍参照上一次的语言报告, 文字写出来. 
        + 第三遍改写英文正确语法.
        + 第四遍翻译成中文, 这是就知道有什么没有表达出来. 
        + 第五遍, 将中文表达出, 而英文的没有的, 增补进英语部分. 
      + 对当时所想的各种词汇/语法都真正的搞清楚. 
      + 以上方法是英语900句, 走遍天下都不怕
      + 语言结构各不同, 日常语法那几种,固化自己文风语调
  - 给 Alex 展示什么
      + 自由的自学之后, 来新西兰的变化, 证明自己学习能力和学习效率
